As respostas para as perguntas mais comuns sobre o SalusVita Tech.

Não. O SalusVita Tech é 100% web — funciona em qualquer navegador moderno (Chrome, Edge, Firefox, Safari). Basta acessar o endereço do sistema. Funciona inclusive em tablets na beira do leito.

O sistema requer conexão com a rede do hospital para funcionar em tempo real. Porém, a infraestrutura pode ser instalada on-premise no servidor do próprio hospital, garantindo independência da internet externa.

Utilizamos autenticação JWT com sessões seguras, controle de acesso por perfil, proteção contra brute-force, e todos os dados são armazenados em banco de dados MySQL com backups regulares. O sistema é projetado com princípios de LGPD desde a sua arquitetura.

A implantação básica pode ser feita em 1 a 2 semanas, incluindo configuração do servidor, cadastro de unidades, tipos de dieta e treinamento da equipe. O sistema já vem pré-configurado com as estruturas mais comuns de hospitais.

A arquitetura do SalusVita Tech possui uma API RESTful que permite integração com sistemas de prontuário eletrônico e outros sistemas hospitalares. Integrações específicas podem ser desenvolvidas sob demanda.
